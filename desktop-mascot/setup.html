Push-to-talk · hold a key to talk

Live two-way conversation, 100% free & on-device — Whisper transcribes you, the active personality replies in its own voice. Run verity voice listen --ptt and hold Right-Shift while you talk, release to send (no clipping, no false triggers — ideal with decks running). Change the key with VERITY_PTT_KEY or ~/.verity-harness/ptt-key (e.g. space, z, ctrl_r); plain verity voice listen is hands-free. First run grants Microphone (+ Input Monitoring for push-to-talk). Point it at your real mic with ~/.verity-harness/mic if your default input is Serato.
